UTAU Plugin 拡張ピッチエディタ Version 2.11

■概要
・旧名称は『Mode1用ピッチエディタ』でしたが、Mode2に対応したため名称を変更しました。
・選択された複数の音符を、専用のピアノロール上でまとめて編集することができます。
・フリーハンドでピッチベンドを引く機能や、モジュレーションをピッチベンドに反映する機能があります。

■使い方
・UTAUインストールフォルダにpluginsフォルダを作成し、そこに解凍したpicedit
　フォルダを移動してください。
・Modeボタンにより、編集するモードを切り替えることができます。
・OKボタンを押したときに表示しているモードのデータのみ保存されます。
・Mode1の欠点である、テンポや先行発声を変更した際にピッチの位置がずれる問題の
　対策として、ピッチ曲線復元機能を実装しています。ピッチ曲線がずれた部分全体を選択し、
　プラグイン画面の「ピッチ曲線を復元」ボタンを押すと、ずれが補正されます。
　（補正しきれない場合もあります。）
　この機能は前回のプラグイン実行時にOthers情報として自動でバックアップしたピッチ
　曲線を元に復元するため、一回もプラグインを実行していない部分は補正できません。
　また、前回実行から復元までの間にピッチをプラグイン以外で変更した場合、その変更
　は無効になります。
・Mode2編集中に点を右クリックすると、線のタイプを変えることができます。
・『Mod...』ボタンを押すと、モジュレーションをピッチ曲線に反映することができます。
　この機能は周波数表を使用するため、事前に作成する必要があります。
・試聴ボタンを押すと、入力中のピッチを試聴することができます。
　試聴には、panda氏が作成された"playinplugin.exe"が必要です。以下のリンク先から
　「playinplugin」をダウンロードし、解凍して出てきた"playinplugin.exe"を
　プラグインのフォルダにコピーしてください。
　http://www20.atwiki.jp/utaou/pages/36.html
