UTAU Plugin 拡張フラグエディタ Version 1.11

■概要
　複数の音符のフラグをまとめてGUI上で編集するフラグエディタです。
　音量、モジュレーション、子音速度も同様の操作で編集することができます。

■注意
　このフラグエディタ実行後にフラグが再構成されるため、実行前とフラグの並び
　順が変わることがあります。

■使い方
　・フラグを編集する範囲の音符を選択し、拡張フラグエディタを起動してください。
　・下部のドロップダウンから編集したいフラグを選択してください。
　・グラフエリアをドラッグすることで選択したフラグの編集を行うことができます。
　・グラフエリア下部の数値部をダブルクリックすることで、直接数値を入力できます。
　・歌詞の下のFlagsエリアをダブルクリックすることで、UTAUと同じような操作で
　　フラグの文字列を編集することができます。
　・画面上部のFlag情報エリアをダブルクリックすることで、フラグの最大値、最小値等
　　を設定することができます。
　・画面右端のスピンボタンにより表示スケールが変化します。
　・Max=1, Min=0, Def=0のフラグをOn,Offの状態のみを持つフラグとして扱います。
　・値の設定されないフラグは1として扱います。

■設定ファイル
　プラグインのフォルダにある設定ファイル（flagtype.ini）に各フラグの情報が記載
　されています。これを編集することでフラグの並べ替えや余計なフラグの削除など、
　自由に編集するすることができます。
　設定ファイルを削除すると、全ての設定が初期状態に戻ります。
